Google連携

管理者のみがAPI設定を変更できます。
Google Cloud Platform
Google スプレッドシートやBigQueryと連携するには、GCPプロジェクトの設定が必要です。
設定手順
  1. Google Cloud Console でプロジェクトを作成
  2. 「APIとサービス」→「ライブラリ」から Google Sheets API と BigQuery API を有効化
  3. 「APIとサービス」→「認証情報」→「OAuth 2.0 クライアントID」を作成（種類: ウェブアプリケーション）
  4. 承認済みJavaScriptオリジンに を追加
  5. 作成されたクライアントIDを上のフィールドに貼り付けて保存
